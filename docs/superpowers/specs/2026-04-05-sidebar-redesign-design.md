# Sidebar Redesign — Design Spec

**Date:** 2026-04-05
**Status:** Approved

---

## Overview

Переработка структуры и визуального оформления сайдбара приложения Afina. Цель — упростить навигацию, добавить точку входа для запуска флоу, убрать визуальные разделители между сайдбаром и рабочей зоной.

---

## Структура

### Шапка

- Логотип Afina — фиксированный, не скроллируется вместе с содержимым меню.

### Основная навигация

Вертикальный список пунктов. Каждый пункт — блок: крупная иконка сверху, текстовая подпись снизу.

Порядок пунктов:

1. **Запустить** — первый пункт, отбит дополнительным отступом снизу от остальных пунктов (без линии-разделителя). При нажатии открывает flyout-панель (см. ниже). Визуально является частью навигационного списка, но отделён отступами.

2. **Сигналы** — badge справа с количеством новых/активных сигналов. При нажатии основная рабочая зона переключается на список карточек сигналов.

3. **Кампании** — badge справа с количеством активных или требующих внимания кампаний. При нажатии основная рабочая зона переключается на список карточек кампаний.

4. **Статистика** — при нажатии основная рабочая зона переключается на дашборд с таблицей и визуализацией.

### Flyout-панель «Запустить»

- Выезжает **справа от сайдбара**, поверх основного контента (overlay, не сдвигает рабочую зону).
- Закрывается: крестик (X) в правом верхнем углу панели + клик вне области панели.
- Содержит **два раздела** с заголовками и карточками действий:

  **Запустить поиск сигналов**
  Карточки сценариев: реактивация, удержание и др. (контент карточек определяется отдельно).

  **Запустить новую кампанию**
  Карточки типов кампаний, включая системные шаблоны и создание с нуля.

### Подвал

Фиксированный, не скроллируется.

- **Баланс** — текстовое отображение текущего баланса (₽ XXX).
- **Аватар + имя пользователя** — кнопка-триггер для dropdown-меню. Dropdown содержит три пункта в следующем порядке:
  1. Настройки
  2. Финансы
  3. Выйти

## Визуальное оформление

| Параметр | Значение |
|---|---|
| Ширина сайдбара | 200–250px |
| Фон сайдбара | Совпадает с фоном рабочей зоны |
| Граница сайдбара | Отсутствует (border-r убирается) |
| Разделение секций | Только отступами (margin/padding), без линий и разделителей |
| Пункты навигации | Иконка (крупная) сверху, текст-подпись снизу — вертикальный список |
| Hover-эффект | Стандартный для sidebar-accent |

---

## Что убирается

- `border-r` между сайдбаром и рабочей зоной.
- Секция «Недавние» (список последних кампаний).
- Пункт «Финансы» из основной навигации (переехал в аккаунтный dropdown).
- Горизонтальная компоновка пунктов навигации (иконка слева, текст справа).
- Separator-компонент между балансом и аккаунтом.
- Пункты «Управление аккаунтом» и «Настройки» дублировались — теперь один dropdown с тремя чёткими пунктами.

---

## Вне scope

- Визуальное оформление flyout-панели (карточки, типографика, сетка) — отдельная задача.
- Содержимое карточек в разделах «Запустить поиск сигналов» и «Запустить новую кампанию».
- Страница Финансов (доступна через аккаунтный dropdown, реализация позже).
