# Стартовый экран и онбординг — финальная конфигурация

**Статус:** Готово к передаче в дизайн
**Дата:** 29.04.2026
**Связанные документы:** anketa-konfiguratsiya.md

---

## Состав экрана

- Welcome-блок с тремя карточками-шагами (текущая структура прототипа сохраняется).
- Чат с тремя заскриптованными подсказками-вопросами.
- Промпт-бар внизу.

## CTA-кнопка

- Расположение: внутри карточки **«Шаг 1»**, после текста описания.
- Это главная точка входа в платформу — она убирает равноправность элементов и задаёт пользователю чёткий путь.

## Поведение при нажатии

Зависит от того, проходил ли пользователь анкету раньше.

- **Анкета пройдена ранее** → нажатие запускает флоу создания сигнала.
- **Анкета пропущена** → нажатие запускает анкету. После её завершения происходит плавный переход во флоу создания сигнала, без возврата на стартовый экран.

## Чат и подсказки

- Welcome-экран и три подсказки в чате сохраняются как механизм объяснения системы.
- Чат не делает навигацию — отвечает в самом чате.
- Логика реакции на свободный ввод вне трёх подсказок — за рамками прототипа.
