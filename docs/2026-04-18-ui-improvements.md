# Список задач: Исправление интерфейса

## 1. Навигация и боковые меню
* **Очистка UI:** Удалить заголовок «Запустить» в открывающихся меню.
* **Очистка UI:** Убрать разделитель под строкой поиска.
* **Исправление поиска:** Добавить стандартный отступ (padding-left) для иконки поиска. Сейчас она прижата к левому краю Input.
* **Обновление текстов:** Под заголовком «Новая коммуникационная кампания» добавить уточняющий текст: «Новая коммуникационная кампания по уже существующим сигналам по готовым сигналам».

## 2. PromptBar (Строка ввода и управления)
* **Позиционирование:** Прибить PromptBar к нижней границе вьюпорта (убрать текущий отступ в 3% снизу).
* **Контекстные Placeholder:** Реализовать динамическую смену текста-подсказки:
    * В разделе «Сигналы»: предлагать помощь с поиском нужных сигналов.
    * В разделе «Кампании»: предлагать помощь в поиске нужной кампании по параметрам.
* **Логика дополнения:** При клике на ноду или выборе параметров текущий текст в PromptBar не должен затираться. Новые данные должны дополнять уже введенный контент.
* **Интерфейс параметров:** Реализовать панель управления нодой, которая выезжает вверх прямо из PromptBar (по аналогии с выбором вариантов в Claude). Окно должно содержать набор внутренних параметров ноды.

## 3. Workflow (Канвас графа)
* **Стиль связей:** Исправить отображение подписей на линиях (линии workflow) — сейчас текст не читается.
* **Масштабирование:** Растянуть область канваса на всё доступное пространство. Чат (PromptBar) должен отображаться слоем поверх канваса.
* **Навигация по канвасу:** Реализовать перемещение (panning) левой кнопкой мыши (инструмент «Рука») без необходимости зажимать пробел.
* **Унификация нод:** * Переименовать все ноды в **существительные на русском языке**.
    * Привести все ноды к единому визуальному стилю и структуре.
    * Прописать нодам внутренние параметры.

## 4. Состояния и управление кампанией
* **Индикация:** В заголовке запущенной кампании выводить цветовой индикатор статуса.
* **Динамическая матрица кнопок:**
    * **Статус «Черновик» (не запущена):** Кнопки `[Сохранить черновик]` и `[Запустить]`.
    * **Статус «Запущена»:**
        * Заменить «Сохранить черновик» на кнопку `[Посмотреть статистику]`.
        * Заменить «Запустить» на кнопку `[Остановить]`.
        * Добавить третью кнопку: `[Дублировать кампанию]`.
